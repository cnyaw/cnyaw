## 🏗️ 核心技術生態系 (The Smallworld Ecosystem)

我的所有開發工作都建立在長期積累的模組化基石之上：

* **[sw2 (Smallworld2)](https://github.com/cnyaw/sw2)**: 萬物之基。一個高效的跨平台網路引擎，支援 Bigworld 分散式架構，處理節點通訊、狀態同步與高併發連線。
* **[good (Generic Object Oriented Design)](https://github.com/cnyaw/good)**: 表現層基石。自主研發的跨平台 2D 遊戲引擎與編輯器，定義了物件化渲染與資源管理的標準。
* **[Nature Online](https://smallworld.idv.tw/game/nature/)**: 集大成之作。基於 sw2 與 good 構建的大規模分散式模擬平台，整合了生命遊戲、遺傳算法與 Boids 鳥群算法，實現成千上萬節點的即時同步。

## 🧠 現代實驗與研究中 (Current Focus)

目前我正將過去的系統開發經驗，延伸至 **AI Agentic Workflows** 與 **個人知識管理**：

* **KDP (Knowledge Distillation Process)**: 開發中的知識蒸餾管線，利用 LLM 與知識圖譜將非結構化文本轉化為可檢索、可演化的知識節點。
* **Vibe Coding & Autonomous Agents**: 探索如何將分散式節點邏輯與 Local LLM (如 Ollama) 結合，建構具備數據主權與自我進化能力的 Agent 系統。
* **實用 AI 工具**: 如「窮人的 AI：自動漫畫分鏡切割 (OpenCV.js)」，致力於將複雜技術轉化為解決生活問題的輕量化方案。
