## 🏗️ The Smallworld Tech Stack (Core Ecosystem)

My projects are built on a foundation of long-term, modular accumulation:

* **[sw2 (Smallworld2)](https://github.com/cnyaw/sw2)**: The bedrock. A high-performance, cross-platform network engine supporting the **Bigworld** distributed architecture. It handles node communication, state synchronization, and high-concurrency connections.
* **[good (Generic Object Oriented Design)](https://github.com/cnyaw/good)**: The visual engine. A custom-built 2D game engine and editor that standardizes object-oriented rendering and resource management.
* **[Nature Online](https://smallworld.idv.tw/game/nature/)**: The culmination. A massive distributed simulation platform built on sw2 & good. It integrates **Conway's Game of Life**, **Genetic Algorithms**, and **Boids (Flocking)** for real-time synchronization across thousands of nodes.
